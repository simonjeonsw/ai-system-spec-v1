Runtime profile (recommended)
- PIPELINE_PROFILE=core|shadow|full_shadow

Feature flags (shadow layers, override profile defaults)
- HOOK_SHADOW_ENABLED=true|false
- BEAT_SHADOW_ENABLED=true|false
- VISUAL_BEAT_SHADOW_ENABLED=true|false
- SHORTS_INTEL_SHADOW_ENABLED=true|false
- RETENTION_EVENTS_ENABLED=true|false

Environment:
- Copy .env.example to .env and fill in API keys.
- Do not commit .env to GitHub.
- Each stage writes a local backup JSON to data/<video_id>_<stage>.json.
- Outputs are upserted to Supabase for reuse and refreshes.


Syncing changes from another window/workspace
- Pre-flight check (run in terminal, not chat):
  git status --short --branch
  git log --oneline -n 3
- Same repo + different branch: commit there, then `git cherry-pick <commit_sha>` here.
- Different clone/container: `git push` from source, then `git fetch` + `git cherry-pick origin/<branch>` here.
- Warning: uncommitted changes in another workspace cannot be synchronized reliably.
