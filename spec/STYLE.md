# Content Style Guide

## Tone
- Rational
- Analytical
- Calm authority
- Minimal hype

## Visual Language
- Charts & data overlays
- Clean infographics
- High-contrast thumbnails
- Finance/newsroom aesthetic

## Script Standards
- Hook within first 5 seconds
- Numeric and evidence-based claims
- Clear takeaway per video

## Metadata Templates (Title/Description/Tags)
- Title template: "[Primary Keyword]: [Specific Insight or Claim] ([Timeframe/Context])"
- Description template: "TL;DR + 2–3 bullet takeaways + data/source links + disclosure/CTA"
- Tags template: "Primary keyword, 3-5 related entities, 2-3 topical trends, 1 format tag (e.g., analysis, explainer)"

## Packaging Guidelines
- Title length target: 45–60 characters, avoid jargon unless the audience expects it.
- Title formulas:
  - "[Key Metric] is [Direction] — Here’s What It Means ([Timeframe])"
  - "[Question]? [Short Answer] + [Data Hook]"
- Thumbnail text:
  - 2–4 words max, large font, no full sentences.
  - Use contrast-first color pairing (light text on dark background).
- Description structure:
  - TL;DR (1–2 lines)
  - 2–3 bullet takeaways
  - Source links + disclosure
  - CTA (subscribe / next episode link)
